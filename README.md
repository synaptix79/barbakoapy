# Barbakóa Landing

Landing page en Next.js App Router + TypeScript + Tailwind para Barbakóa, Texas Barbecue Paraguay.

## Pendientes antes de publicar

- Reemplazar `NEXT_PUBLIC_WHATSAPP_NUMBER=595XXXXXXXXX` por el número real.
- Confirmar precios reales del menú y cargar `price` en `lib/menu.ts`.
- Confirmar horarios vigentes antes de mostrarlos.
- Confirmar si Monchis y PedidosYa siguen como canales activos.
- Cargar fotos reales de productos solo cuando el cliente las provea.

## Assets

- `public/assets/barbakoa-logo.png`: logo provisto por el cliente, centrado y preparado con fondo transparente para usarse sobre el sitio.

## Desarrollo

```bash
npm install
npm run dev
```

Si no hay Node instalado globalmente, en este workspace quedó descargado Node portátil en `.tools/node`.
